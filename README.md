# BMI-Calculator
BMI 计算器（Toga 跨平台桌面GUI）
 
基于 Python + Toga 框架开发的跨平台桌面版 BMI 计算器，支持 Windows / macOS / Linux 运行，界面简洁轻量化，自带输入异常捕获，零基础可直接运行。
 
✨ 项目特点
 
- 🖥️ 原生桌面GUI：基于 Toga 官方跨平台框架，界面干净美观
- ⚡ 极简操作：输入身高、体重一键计算 BMI 指数
- 📊 智能结果判定：自动区分偏瘦/正常/轻度偏胖/肥胖四种体型状态
- 🛡️ 完善异常处理：拦截空输入、文字输入、身高为0等报错
- 🔤 无乱码问题：内置关闭彩色日志配置，彻底解决 Windows 终端乱码
- 🚀 跨平台兼容：一次编写，全平台可运行

📖 功能使用说明
 
1. 体重输入框：输入纯数字体重，单位：千克(kg)
2. 身高输入框：输入纯数字身高，单位：米(m)
3. 点击 计算BMI 按钮
4. 窗口自动展示：精确两位小数的 BMI 值 + 体型健康建议
 
判定标准（国际通用BMI标准）
 
BMI 数值范围 身体状态 
BMI ＜ 18.5 偏瘦，建议补充营养 
18.5 ≤ BMI ≤ 25 体重正常，保持作息饮食 
25 ＜ BMI ≤ 30 轻度偏胖，控制饮食、适度运动 
BMI ＞ 30 肥胖，需要规律运动、控制体重 
 
🧩 核心代码逻辑解析
 
1. 环境兼容配置
专门适配 Windows 系统，规避中文日志、彩色字符导致的乱码报错。
 
2. UI界面布局
 
采用 垂直弹性布局(COLUMN)，统一外边距，控件整齐排版：
 
- 体重输入框、身高输入框
- 计算功能按钮
- 结果动态展示文本框
 
3. 核心计算逻辑
 
通过公式计算 BMI：
BMI = 体重(kg) \div 身高(m)^2
代码自动保留两位小数，结果更精准。
 
4. 双重异常捕获
 
-  ValueError ：捕获空输入、汉字、符号、字母等非法输入
-  ZeroDivisionError ：捕获身高输入0的非法计算场景
 
❌ 常见报错与解决方案
 
1. 运行提示 no module named toga
未安装依赖，执行： pip install toga 
2. 终端出现乱码字符
代码已内置  TOGA_NO_COLOR  配置，重新运行即可修复
3. 点击按钮无反应
代码函数绑定顺序规范（先定义函数、后创建按钮），复制完整代码即可解决
4. 输入身高0报错
程序已自动拦截并提示： 身高不能填写0
